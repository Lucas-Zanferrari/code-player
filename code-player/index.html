Hello ya'll! I'm Lucas Zanferrari, a web developer, science and technology student (neeeeeerd... :P) and innovation enthusiastic.

This is my first individual project that goes online (yay)! It is a pretty basic code player of HTML, CSS and plain old JavaScript (no external resources support yet) which I coded in roughly one day only to practice and show to potential employers as part of my folio, so please don't over judge me on account of potential bugs and/or lack of features (although ideas and feedback are more than welcome). :)

Also, I'm not a designer (though I'm interested in the field), so don't expect astonishingly great looks from my system. But then again, ideas and feedback are more than welcome hahaha.

You can start to use the system by navigating through the top menu bar, and if you by chance wish to come back to this text at any point, just click in that big "CodePlayer" text on your screen's upper-left side. The Run button will display the result of your code in the OUTPUT frame, however it should not be required in normal circumstances as the code result is displayed in real-time (right as you type it).

For any suggestions, recommendations, criticism and anything you might have to say about my work (or just chitchatting, I'm a welcoming person hahaha), you can find me on lucas.zanferrari@gmail.com or on my Facebook profile.

Cheers!
